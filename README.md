使用room的好处是，可以直接使用数据库，不需要再去写sql语句，直接使用room的api就可以了，
而且room的数据库是自动创建的，不需要手动创建表，而且room的数据库是自动升级的，不需要手动升级表结构，而且room的数据库是自动加密的，不需要手动加密。
